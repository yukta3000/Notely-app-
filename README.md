# Notely — Notes App

A clean, feature-rich notes app built with **vanilla HTML, CSS & JavaScript** — no frameworks, no dependencies.

**Developed by: Yukta Taneja**

---

## Features

- Create notes with a title and content
- 6 accent color options per note
- Pin important notes to the top
- Edit notes via a smooth modal dialog
- Delete notes with one click
- Live search across all notes
- Filter: All / Pinned / Recent (last 7 days)
- Notes persist in localStorage (no backend needed)
- Responsive masonry grid layout

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid)
- Vanilla JavaScript (no libraries)

## Getting Started

Just open index.html in any browser — no build step needed!
